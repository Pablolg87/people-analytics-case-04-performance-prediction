# People Analytics: Employee Performance Prediction

## Overview
This project analyzes the main factors related to low employee performance and builds a predictive model using HR data.

## Dataset
The dataset includes variables such as:

- Age
- Tenure
- Department
- Salary level
- Engagement
- Training
- Absenteeism
- Performance

## Project Steps
- Data cleaning and preprocessing
- Exploratory data analysis
- Feature preparation
- Predictive modeling
- Model evaluation
- HR insights and conclusions
