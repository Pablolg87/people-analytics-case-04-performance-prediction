# People Analytics: Employee Performance Prediction

## Overview
This project analyzes the main factors related to low employee performance and builds a predictive model using HR data.

## Dataset
The dataset includes variables such as:

- Age
- Tenure
- Department
- Salary level
- Engagement
- Training
- Absenteeism
- Performance
  
## Project Steps
- Data cleaning and preprocessing
- Exploratory data analysis
- Feature preparation
- Predictive modeling

## Results

- Model used: Random Forest
- Key metric: Recall for low performers
- Business goal: Early detection of underperformance

## Key Drivers

- Engagement ↓ → Performance ↓
- Absenteeism ↑ → Performance ↓
- Training impact varies by department

## HR Use Case

This model can help HR teams:
- Identify at-risk employees early
- Prioritize interventions
- Optimize training investment
- Model evaluation
- HR insights and conclusions
